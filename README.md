# AidLink

AidLink is a mobile-first resource app that connects users to essential services such as food, shelter, medical support, and financial assistance.

## Features

- Browse local resources by category
- Search and filter resources
- View detailed resource information
- Submit help requests
- Firebase Firestore backend (real-time data)

## Tech Stack

- React Native (Expo)
- TypeScript
- Firebase Firestore
- Expo Router

## Screens

- Home
- Resources
- Resource Details
- Request Help

## How it works

- Resources are fetched from Firestore
- If Firestore is unavailable, mock data is used
- Help requests are saved to Firestore
